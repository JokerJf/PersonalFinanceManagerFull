# Finance Manager

Приложение для управления личными финансами.

## Функции

- Учёт доходов и расходов
- Управление счетами (карты, наличные, банки)
- Отслеживание долгов (я должен / мне должны)
- Управление кредитами и рассрочками
- Планирование бюджета
- Конвертация валют
- Семейный режим

## Технологии

- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeORM
- База данных: PostgreSQL

## Установка

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
